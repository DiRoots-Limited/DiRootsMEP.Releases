# DiRootsMEP

**DiRootsMEP** is a next-generation Revit plugin developed by **DiRoots** to simplify and automate workflows for MEP engineers.  
It provides intelligent tools to manage, validate, and optimize MEP model data — starting with **Insulation Management** and **System Validation** utilities.

This is just the beginning - **many more MEP-focused tools are coming soon** to make your modeling faster, smarter, and more consistent.

---

## 🚀 Current Tools
### 🧊 Insulation Tools
Automate and control the application and visibility of insulation for pipes and ducts.

#### Add Insulation
Automatically apply insulation to pipes and ducts based on predefined rules and project parameters.

**Key Features:**
- Create rules using **Size Range**, **System Type**, **Worksets**, and **Custom Parameters**.  
- Choose from three **selection scopes**:
  - *Whole Model* – applies to all relevant elements.  
  - *Active View* – applies to visible elements only.  
  - *Current Selection* – applies to manually selected elements.  
- Enable **Auto Update** to apply insulation automatically to new elements matching defined rules.  
- **Export / Import Rules** as `.json` files for easy reuse or sharing.  
- Apply all rules instantly across the selected scope.

#### Hide Insulation
Temporarily hide insulation for selected pipes or ducts without deleting it.  
- Ideal for model review, coordination, or presentation.  
- Quickly toggle insulation visibility for a cleaner modeling view.

---

### ⚙️ System Validation
Ensure MEP data accuracy and consistency through automated validation checks.

#### Undefined System Type
Identify and highlight pipes and ducts that have an **undefined system type** within the active view.  
- Detects unassigned or inconsistent system data.  
- Helps maintain model integrity and coordination accuracy.
